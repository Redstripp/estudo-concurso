# Checklist de Publicacao

Use este passo a passo antes de publicar o sistema.

1. Verificar se `js/config.js` existe.
2. Rodar `npm run check:js`.
3. Rodar `npm test`.
4. Testar login.
5. Testar cadastro.
6. Testar logout.
7. Testar `app.html`.
8. Verificar console do navegador.
9. Verificar se nao ha erro 404.
10. Verificar se nenhuma chave secreta foi publicada.
11. Testar pelo GitHub Pages, nao apenas localmente.
12. Registrar a alteracao no historico.
